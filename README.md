# Задание 1. Seq.map

## Задача 1

### Текст задачи

На основе заданной последовательности чисел получить последовательность из значений true и false. True
— если соответствующий элемент исходной последовательности чётный, и false — в
противном случае

### Алгоритм решения

1) Запрашиваем у пользователя количество чисел, входящих в исходную последовательность.
2) С помощью функции проверяем, чтобы количество, которое он запросил, было натуральным числом.
3) Запрашиваем числа последовательности с пользователя (проверяем функцией на целое).
4) Выводим исходную последовательность.
5) С помощью Seq.map создаём вторую последовательность так, что все чётные элементы первой - true, нечётные - false.
6) Выводим получившуюся последовательность.

# Задание 2. Seq.fold

## Задача 1

### Текст задачи

Последовательность содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов последовательности.

### Алгоритм решения

1) Запрашиваем у пользователя количество чисел, входящих в исходную последовательность.
2) С помощью функции проверяем, чтобы количество, которое он запросил, было натуральным числом.
3) Запрашиваем римские цифры, которые входят в последовательность с пользователя (проверяем функцией на корректность ввода).
4) Выводим исходную последовательность.
5) С помощью функции с использованием Seq.map создаём вторую последовательность, меняя римские цифры на арабские.
6) С помощью Seq.fold считаем сумму элементов последовательности, поочерёдно складывая цифры из неё.
7) Выводим сумму элементов последовательности.

# Задание 3. Решить с использованием последовательности задачу на обработку файлов/каталогов

## Задача 1

### Текст задачи

Вывести количество файлов, имя которых начинается с заданного
символа, в указанном каталоге и его подкаталогах.

### Алгоритм решения

1) Запрашиваем у пользователя путь к каталогу.
2) С помощью рекурсивной функции создаём последоватльность из имён файлов каталога.
3) Запрашиваем у пользователя символ.
4) С помощью кортежа из Seq.filter (записывает в последовтальность имена файлов с заданного символа без путей) 
и Seq.length подсчитываем количество файлов.
5) Выводим количество файлов, которые наичнаются с заданного символа.
